# CryptoXY

A website for a crypto trading company, featuring a crypto calculator, 
customer reviews, and a contact form.

## 🌐 Live Demo
[https://nikola-jovanovic-1.github.io/CryptoXY/](https://nikola-jovanovic-1.github.io/CryptoXY/)

## 📌 Features
- Crypto calculator (buy/sell prices)
- Customer reviews with slider
- Contact form with validation
- Responsive design

## 🛠️ Technologies
- HTML5
- CSS3
- JavaScript
- Bootstrap
